You are on the Grid page of the Lazy's Framework !

2 types of classes are available to manage the design in a responsive way : col and offset
So we can choose according to the size of the device the size of the different elements thanks to the classes and therefore change the structure of our application

/!\ YOU MUST USE ALL THE THREE TAGS OF THE EACH TYPE IF YOU WANT TO HAVE A RESPONSIVE DESIGN
Ex : <div class="col-lg-1 col-md-4 col-sm-12 offset-md-4">
n = number of columns in the grid
scree width class name
width › 1000px col-lg-n
600px ‹ width ‹ 1000px col-md-n
width ‹ 600px col-sm-n
