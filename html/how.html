How to use Lazy's Framework ?

It is possible to use the framework in CSS or SCSS. In CSS, you only
need to import the CSS folder containing the CSS file of the framework named
"Lazy.css" . Upload this file to your html and voila!

Regarding the use of the scss, just follow the tutorial below :)

SCSS

Download SCSS folder to the github repository :
https://github.com/Quinou-kun/Lazy-s-Framework
Import the "Lazy.scss" file into your personal CSS file :
@import "Lazy";
Use Lazy's Framework's mixins to create custom objects with your custom classes :
.myCustomClass {@include genericLazyMixin(); ...}
For example, create a custom grid :
.wrapper{@include wrapper; @include construct-grid($gridBackground, $gridHeight, $gridColumns); your custom properties}
